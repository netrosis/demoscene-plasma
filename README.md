A really simple implementation of the 1990s plasma effect commonly seen in the Amiga and DOS demoscenes, based on the [GLSL implementation by TriggerHLM](https://www.shadertoy.com/view/MdXGDH) and using the MacOS AppKit library for window management.

The primary purpose of this demonstration was to see how small I could make an application that 1.) Opens a window, and 2.) Displays an animated bitmap texture. Current personal record is 257 lines of code compiled into a 71kb binary.
